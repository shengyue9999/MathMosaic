# MathMosaic - 数绘 · ShuHui

**把小学数学，拼成一本会发光的绘本**

一个用**绘本风格 + 拼图形式**呈现的小学数学知识系统。  
每一章都是一块可以拖拽、可以点亮、可以连接的梦幻拼图卡片，帮助孩子在“玩”的过程中建立知识地图、发现知识关联、找到自己的短板。

---

### ✨ 项目特色

- 温暖手绘风插画 + 柔和绘本色系
- 知识以“拼图板块”形式呈现，完成即上色发光
- 清晰的知识依赖关系图（看到哪块没亮，就知道哪里没学好）
- 支持亲子共学，家长可轻松看到孩子掌握情况
- 数据完全开源，使用 YAML 格式，老师和家长可自行补充修改
